#########
PyrepBUFR
#########

A pure Python implementation WMO BUFR decoder that can handle NWS/NCEP prepbufr formatted files.

Prequisites
-----------

None outside of the Python standard packages
 - collections
 - copy
 - math
 - sys
 - typing
 - textwrap
 - xml
   - dom
   - etree
Numpy will be used if present.

-----------
